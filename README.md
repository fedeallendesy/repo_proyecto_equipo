# Proyecto Transporte Público Santiago

Este repositorio contiene el trabajo de la Entrega 01 y la Entrega 02 del análisis de la oferta de buses en Santiago y su relación con la demanda de pasajeros.

## Estructura del repositorio
- `propuesta/`: documentación general del proyecto.
- `proyecto/Entrega_01/`: primera entrega individual y documentación asociada.
- `proyecto/Entrega_02/`: segunda entrega con bases de datos limpias, scripts y documentación de análisis.

## Contenido principal
- `proyecto/Entrega_02/README.md`: descripción, hipótesis y avance del proyecto para la segunda entrega.
- `proyecto/Entrega_02/Allendes_integrante_02_Federico_base/`: base limpia y documentación del segundo integrante.
- `proyecto/Entrega_02/Poblete_Integrante_01_Makarena_database_01/`: datos originales, scripts de limpieza y diccionario de variables.

## Objetivo del proyecto
Validar si la frecuencia de buses en Santiago está alineada con los puntos de mayor demanda y detectar posibles desigualdades en la distribución del servicio.

## Cómo usar este repositorio
1. Abra `proyecto/Entrega_02/README.md` para revisar el avance y la metodología.
2. Revise los archivos de datos y el notebook en `proyecto/Entrega_02/Poblete_Integrante_01_Makarena_database_01/` para verificar el proceso de limpieza.
3. Consulte `proyecto/Entrega_02/Allendes_integrante_02_Federico_base/` para el resultado de la base limpia y la documentación técnica.
